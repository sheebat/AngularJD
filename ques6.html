ng-app

The ng-app directive tells AngularJS that the div element is the "owner" of an AngularJS application.

ng-model

The ng-model directive binds the value of the input field to the application variable name.

ng-bind

The ng-bind directive binds the innerHTML of the p element to the application variable name.

ng-init

The ng-init directive initializes AngularJS application variables.

The ng-switch directive lets you hide/show HTML elements depending on an expression. Child elements with the ng-switch-when directive will be displayed if it gets a match, otherwise the element, and its children will be removed. You can also define a default section, by using the ng-switch-default directive, to show a section if non of the other sections get a match.
